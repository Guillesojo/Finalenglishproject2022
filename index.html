The B1 English Guide

As students of a foreign language, we sometimes have questions about a topic, word, or strategy. That's why in this project, we created an easy way to respond to these common questions. Here you'll encounter information about various topics and some resources to help you learn further. Sometimes it can be difficult to search for quality learning material on the internet, so we hope this project will help you have a better learning experience.
