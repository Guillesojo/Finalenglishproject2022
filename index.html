The B1 English Guide

As students of a foreign language, we sometimes have questions about a topic, word, or strategy. That's why in this project, we created an easy way to respond to these common questions. Here you'll encounter information about various topics and also some resources to help you learn further. We know sometimes it's hard to search for quality material on the internet, so we hope this project will help you to have a better learning experience!

Simple Past Tense

We use simple past to talk about completed actions and things that existed before now.

Simple Present Tense

We use simple present to describe habits, facts, actions happening now, permanent actions and plans.

Past Progressive

We use past progressive to describe an action that was in progress at a specific time in the past.

Present Progressive

We use present progressive to describe an action that is in progress right now.

Future Progressive

We use future progressive to describe an action that will be in progress at a future time.

Past Perfect Tense

We use past perfect when we want to talk about actions that were completed before some point in the past.

Present Perfect Tense

We use present perfect when we want to talk about an event that happened at an unspecified time in the past.

Future Perfect Tense

We use future perfect for actions that will be completed before some other point in the future.
