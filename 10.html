Zero and First Conditionals


Zero conditional: We use this conditional when the result of an action will always happen. We also use this conditional with facts.
Example: If you put water in the freezer, it turns into ice.
Formula: If + present simple, ... present simple.

First conditional: We use the first conditional to describe things that may happen in the future or present.
Example: If it's sunny, weâ€™ll go to the park.
Formula: If + present simple, ... will + infinitive
